Kenapa Bab Ini Penting?

Database itu bukan cuma tempat nyimpan data, tapi:

  • Penjaga kebenaran data
  • Penentu kecepatan sistem
  • Penyangga keputusan bisnis
Banyak sistem gagal bukan karena kodenya, tapi karena fondasi databasenya salah sejak awal.

1.1 Konsep Dasar Database

Sebelum masuk ke SQL, MySQL, query, dan lain-lain, kita harus menyamakan cara berpikir dulu.

a. Apa itu Database?

Definisi Sederhana

Database adalah kumpulan data yang disimpan secara terstruktur agar bisa diakses, dikelola, dan dianalisis dengan efisien.

Kata kunci penting:

  • Kumpulan data
  • Terstruktur
  • Bisa diolah

Kenapa Kita Perlu DBMS (Bukan File Biasa)?

Masalah jika pakai file:

  • Data dobel
  • Data bentrok
  • Tidak atomic (setengah masuk)
  • Tidak aman

DBMS Menyelesaikan Ini Dengan:
  • Transaction
  • Constraint
  • Locking
  • Index
  • Role & Permission
